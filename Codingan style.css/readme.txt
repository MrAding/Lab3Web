cara membuka file :
1. klik kanan pada file "style.css"
2. pilih menu "Open With", kemudian pilih "Visual Code Studio"
3. anda bisa meletakkan codingan ini di bagian <head> / <body>
4. untuk menjalankan file nya anda harus membuat codingan baik judul/isi <h1>/<h2>
5. jika sudah membuat file nya dan ingin melihat ouput maka anda bisa menekan tombol "CTRL+F5 / F5" ataupun anda bisa melakukan run & debug secara manual di menu "run" pada visual code studio
